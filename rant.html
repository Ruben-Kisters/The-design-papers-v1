Ik ben kapot

Ik heb DAGEN aan deze rotwebsite besteed, Code gefixed, Bugs gepatched, Bijna haren uitgetrokken.

En dan om waarschijnlijk afgekeurd te worden.

Waarom?

Waarom moet ik dit doen? Waarom kan ik niet gewoon een normaal artikel schrijven?

Waarom moet ik lijden voor een website die niemand gaat bezoeken?

De strijd

Elke keer als ik dacht dat ik klaar was, kwam er weer een nieuwe bug opdagen.

Elke keer als ik dacht dat ik het begreep, veranderde de opdracht weer.

De hoop

Misschien zal ik ooit terugkijken op deze ervaring en lachen.

De realiteit

Maar voor nu, ben ik gewoon kapot.

Slotwoord

Als je dit leest, hoop ik dat je begrijpt wat ik doormaak.

En als je ooit een website moet maken, wens ik je veel succes — je zult het nodig hebben.

Tot ziens,

Een uitgeputte webontwikkelaar
